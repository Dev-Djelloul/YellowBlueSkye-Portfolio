DÉTAILS DU PROJET

Objectif

L'objectif de ce projet était de dynamiser une page web statique avec JavaScript en ajoutant un carrousel d'images interactif. Ce projet marque mes premiers pas dans le développement JavaScript, avec l'implémentation d'interactions utilisateur simples mais efficaces.

Technologies utilisées

Ce projet utilise HTML5 et CSS3 pour la structure et le style de base du site, et JavaScript pour ajouter des fonctionnalités dynamiques comme le carrousel d'images avec navigation par flèches et par points. J'ai également implémenté des animations et transitions pour améliorer l'expérience utilisateur.

Défis et solutions

Le principal défi était d'implémenter un carrousel fonctionnel avec une navigation fluide et intuitive, tout en gérant correctement les transitions entre les images. Pour résoudre ce problème, j'ai créé des fonctions JavaScript modulaires et j'ai utilisé des événements pour gérer les clics sur les flèches de navigation et les indicateurs de position, en mettant en place une logique de défilement circulaire.

PROJET RIDING CITIES

RIDING CITIES

Riding Cities est le tout premier projet de site web utilisant les languages de bases HTML et CSS et publié sur Github-pages. Il a été conçu lors de mon apprentissage des languages de bases avec l'école Openclassrooms.

HTML5 CSS3

DÉTAILS DU PROJET

Objectif

L'objectif de ce projet était de créer un site web simple et informatif pour une association de skateboard. Il s'agissait de mettre en pratique mes premières connaissances en HTML et CSS, tout en respectant une maquette fournie par le client.

Technologies utilisées

Pour ce projet, j'ai utilisé HTML5 pour structurer le contenu de la page et CSS3 pour le style et la mise en page. Le site est hébergé sur GitHub Pages pour un accès facile.

Défis et solutions

Le principal défi était de créer une mise en page responsive qui s'adapte à différentes tailles d'écran. J'ai appris à utiliser les media queries et les unités relatives pour garantir que le site reste utilisable et esthétique sur tous les appareils.

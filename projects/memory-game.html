MEMORY GAME

Memory Game est un jeu de mémoire créé avec HTML, CSS et JavaScript, et publié sur GitHub Pages. Il a été conçu dans le cadre de ma formation avec Studi pour maîtriser les fondamentaux du développement web.

DÉTAILS DU PROJET

Objectif

L'objectif de ce projet était de créer un jeu de mémoire interactif pour démontrer mes compétences en JavaScript. Le jeu devait inclure un système de score, un minuteur et des niveaux de difficulté variables.

Technologies utilisées

Ce projet a été construit en utilisant HTML5, CSS3 pour la mise en page et le style des cartes et du plateau de jeu, et JavaScript pour toutes les fonctionnalités interactives. J'ai implémenté la gestion des événements pour le retournement des cartes, la validation des correspondances et la gestion de l'état du jeu.

Défis et solutions

Le défi principal était de créer une animation fluide pour le retournement des cartes et de gérer l'état du jeu tout au long de la session. J'ai résolu ce problème en utilisant des transitions CSS pour les animations et en construisant un système robuste de gestion d'état en JavaScript pour suivre les cartes retournées, les correspondances et la progression du jeu.

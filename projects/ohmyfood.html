OHMYFOOD

Ohmyfood est un site web mobile-first avec des animations CSS avancées. Ce projet a été développé dans le cadre de ma formation avec Openclassrooms pour apprendre à créer des animations CSS modernes et attrayantes.

HTML5 CSS3 Animations SASS
Voir le projet en ligne Voir sur GitHub

DÉTAILS DU PROJET

Objectif

L'objectif était de développer un site "mobile first" qui répertorie les menus de restaurants gastronomiques et permet aux utilisateurs de composer leur menu pour réduire leur temps d'attente au restaurant. Le site devait inclure des animations sophistiquées en CSS.

Technologies utilisées

Pour ce projet, j'ai utilisé HTML5 et CSS3 avec le préprocesseur SASS pour organiser mon code. J'ai créé des animations complexes comme des loaders, des effets de survol et des transitions entre les pages en utilisant uniquement du CSS.

Défis et solutions

Le principal défi était de créer des animations fluides et élégantes qui fonctionnent sur tous les navigateurs. J'ai implémenté des keyframes CSS pour des animations personnalisées, et j'ai utilisé des variables CSS pour maintenir une cohérence visuelle. Une attention particulière a été portée à l'optimisation des performances pour assurer une expérience utilisateur fluide.

SOPHIE BLUEL

Le portfolio de Sophie Bluel est un site web responsive construit avec HTML, CSS, JavaScript, et la gestion des dépendances via Node Package Manager. Ce projet implique la création d'un site dynamique permettant de présenter et gérer une galerie de travaux photographiques.

HTML5 CSS3 JavaScript NPM API
Voir le projet en ligne Voir sur GitHub

DÉTAILS DU PROJET

Objectif

L'objectif était de créer une interface frontend pour le portfolio d'une architecte d'intérieur, en intégrant une API backend existante. Le site devait permettre l'affichage des projets, la connexion administrateur, et l'ajout/suppression de projets via une interface sécurisée.

Technologies utilisées

J'ai utilisé HTML5 et CSS3 pour la structure et le style du site, JavaScript vanilla pour l'interactivité et les appels API, ainsi que NPM pour la gestion des dépendances. Le site inclut également une validation des formulaires côté client et la gestion des droits d'accès (JWT).

Défis et solutions

Le principal défi était d'implémenter un système d'authentification sécurisé et une interface d'administration intuitive pour la gestion des projets. J'ai créé une solution modulaire avec séparation des préoccupations (modèles, vues, contrôleurs) et mis en place un système de filtrage dynamique des projets par catégorie.
